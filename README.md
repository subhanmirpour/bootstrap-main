# Project Setup Guide

## Prerequisites

1. **Install Node.js**  
   Ensure you have Node.js installed on your system. You can download it from [nodejs.org](https://nodejs.org/).

## Installation Steps

1. **Install Gulp CLI Globally**  
   ```bash
   npm install --global gulp-cli


2. **Install packages(go to the root directory)**  
   ```bash
   npm install

3. **host the website**  
   ```bash
   npm run dev
